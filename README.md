# ZCG Investments · Disposition Hub

Interactive proposal demo for the **ZCG Investments Disposition Hub** — a Podio-first automation portal for wholesale real estate dispositions.

**Live demo:** https://jakubgoscinny-creator.github.io/zcg-dispo-hub-demo/

Single-file, self-contained HTML artifact. Desktop target: 1440px.

## Modules

1. **Deal Board** — Kanban pipeline (Under Contract · Campaigns Live · Walkthrough · Closing) with offer velocity sparklines, click-through deal drawer, full offer history and 8-step workflow timeline
2. **Buyer Database** — Segmented buyer list with live GetResponse / SimpleTexting / InvestorLift sync status, search, add-buyer modal
3. **Campaign Builder** — Live preview that regenerates email subject, SMS body, and InvestorLift listing copy per selected property · Generate Drafts animation · full draft cards with realistic copy
4. **Workflow Engine** — 8-step automated dispo pipeline, live 40-hour and 96-hour countdown timers, system alert feed
5. **AI Underwriting** — ChatARV.ai integration mockup · ARV + 3 comps + renovation estimate + suggested offer range · push-to-Podio CTAs

## Design

- Bloomberg-terminal meets editorial — dark parchment with warm gold and electric teal accents
- Dark / light theme toggle (preference persists in localStorage)
- Bebas Neue headings · Lato body · JetBrains Mono numerics
- Live ticker tape, count-up stat animations, staggered card reveals
- Every data field has a 3-line hover tooltip explaining Data Source / Integration / Stack

## Tech

Zero build step. Single HTML file. Google Fonts + Font Awesome CDN. Pure vanilla JS.
